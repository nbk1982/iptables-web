iptables Command Reference

Quick reference for common iptables/ip6tables operations, including filtering, NAT, and troubleshooting basics.

Core Concepts

  • Tables: raw, mangle, nat, filter
  • Chains: INPUT, OUTPUT, FORWARD, PREROUTING, POSTROUTING
  • Rules: match conditions plus a target action

Command Pattern

iptables [-t table] COMMAND [chain] [match options] [-j target]
OptionDescription
-tSelect table (default filter)
-L -n -v --line-numbersList rules with detailed numeric output
-A/-I/-D/-RAppend/Insert/Delete/Replace rule
-F/-Z/-XFlush rules / zero counters / delete custom chain
-PSet default policy on built-in chains
-mUse match extension module
-jSet target action

Examples

List Rules

iptables -L -n -v --line-numbers
iptables -t nat -L -n -v
ip6tables -t filter -L INPUT -n

Set Default Policies

iptables -P INPUT DROP
iptables -P FORWARD DROP
iptables -P OUTPUT ACCEPT

NAT

iptables -t nat -A POSTROUTING -s 192.168.0.0/24 -o eth0 -j SNAT --to-source 203.0.113.10
iptables -t nat -A POSTROUTING -s 10.10.0.0/16 -o ppp0 -j MASQUERADE
iptables -t nat -A PREROUTING -d 203.0.113.10/32 -p tcp --dport 2222 -j DNAT --to-destination 192.168.0.10:22

Troubleshooting

  • Check backend mode with iptables -V (nft vs legacy).
  • Confirm required modules (for example nf_conntrack) are loaded.
  • Verify rule order when behavior is unexpected.
  • Use LOG targets and system logs for debugging.
